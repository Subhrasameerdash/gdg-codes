    The equation of a line in slope-intercept form is y = mx + c, where m is the slope and c is the y-intercept.  In this case, m = tan θ = 1/3 and c = 2.

    Therefore, the equation of the first line is:  `y = (1/3)x + 2` which can be rewritten as `x - 3y + 6 = 0`.

Let's analyze the given information to find the slope 'm' of line L1 and its equation.

**1. Understanding Perpendicular Lines and Slopes:**

A crucial concept here is that if two lines are perpendicular, the product of their slopes is -1.  That is, if line L1 has slope 'm', and line L2 is perpendicular to L1, then the slope of L2 (let's call it m2) is given by:

m * m2 = -1   =>   m2 = -1/m

**2. Finding the Slope of L2:**

We know L2 intersects the x-axis at x = 3. This means L2 passes through the point (3, 0).  Since L2 is perpendicular to L1, we can use the information to find the slope of L1. However, we need one more point on L2.
We know that line L1 passes through the point A(1,5). Let's call the point where line L2 intersects line L1 as B(x,y). So, line L2 passes through B(x,y) and (3,0). Thus, the slope of L2 would be (y-0)/(x-3) = y/(x-3).

3.  A point P moves in a plane such that the sum of the absolute distances from P to the lines x + y = 0 and x - y = 0 is always equal to 4. Describe the geometrical shape formed by the path of point P, and provide its defining equation(s).
Sample Answer:
Okay, here's a sample answer to the given assignment question, tailored for a CBSE Class 11-12 student, based on the provided NCERT chapter summary and examples:

**Answer:**

Let P(x, y) be any point in the plane.  We are given that the sum of the absolute distances from P to the lines  x + y = 0 and x - y = 0 is always equal to 4.

Let's find the distance from P(x, y) to each of these lines using the formula for the perpendicular distance from a point (x₁, y₁) to a line Ax + By + C = 0, which is:

Distance = |Ax₁ + By₁ + C| / √(A² + B²)

*   **Distance from P(x, y) to the line x + y = 0:**

    Distance₁ = |x + y| / √(1² + 1²) = |x + y| / √2

*   **Distance from P(x, y) to the line x - y = 0:**

    Distance₂ = |x - y| / √(1² + (-1)²) = |x - y| / √2

According to the problem, Distance₁ + Distance₂ = 4.  Therefore:

|x + y| / √2  +  |x - y| / √2 = 4

Multiplying both sides by √2, we get:

|x + y| + |x - y| = 4√2

Now, we need to consider different cases based on the signs of (x + y) and (x - y) to remove the absolute value signs.  There are four possible cases:

**Case 1: x + y ≥ 0 and x - y ≥ 0**

In this case, |x + y| = x + y and |x - y| = x - y.  The equation becomes:

x + y + x - y = 4√2

2x = 4√2

x = 2√2

Since x + y ≥ 0 and x - y ≥ 0, we have y ≥ -x and y ≤ x.  Therefore, -2√2 ≤ y ≤ 2√2

**Case 2: x + y ≥ 0 and x - y < 0**

In this case, |x + y| = x + y and |x - y| = -(x - y) = y - x.  The equation becomes:

x + y + y - x = 4√2

2y = 4√2

y = 2√2

Since x + y ≥ 0 and x - y < 0, we have x ≥ -y and x < y. Therefore -2√2 ≤ x ≤ 2√2

**Case 3: x + y < 0 and x - y ≥ 0**

In this case, |x + y| = -(x + y) = -x - y and |x - y| = x - y.  The equation becomes:

-x - y + x - y = 4√2

-2y = 4√2

y = -2√2

Since x + y < 0 and x - y ≥ 0, we have x < -y and x ≥ y. Therefore -2√2 ≤ x ≤ 2√2

**Case 4: x + y < 0 and x - y < 0**

In this case, |x + y| = -(x + y) = -x - y and |x - y| = -(x - y) = y - x.  The equation becomes:

-x - y + y - x = 4√2

-2x = 4√2

x = -2√2

Since x + y < 0 and x - y < 0, we have y < -x and y > x. Therefore -2√2 ≤ y ≤ 2√2

**Geometric Shape:**

The equations x = 2√2, y = 2√2, x = -2√2, and y = -2√2 represent four straight lines. Combining these lines, we can see that the path of point P forms a **square**.  The vertices of the square are (2√2, 2√2), (-2√2, 2√2), (-2√2, -2√2), and (2√2, -2√2).  The square is centered at the origin and its sides are parallel to the coordinate axes.

**Conclusion:**

The geometrical shape formed by the path of point P is a **square** with vertices (2√2, 2√2), (-2√2, 2√2), (-2√2, -2√2), and (2√2, -2√2). This square can also be represented by the following equations:

* x = ±2√2
* y = ±2√2
